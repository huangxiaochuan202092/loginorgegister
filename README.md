# 项目介绍
本项目实现了登录即注册的功能，并在此基础上开发了用户功能模块。

## 项目操作步骤
1. **克隆项目**：
使用`git clone`克隆项目地址。
2. **修改配置**：
修改以下配置：
    - Redis和MySQL的连接配置。
    - 邮箱及邮箱授权码。
3. **初始化项目**：
执行`go mod tidy`初始化项目。
4. **运行项目**：
修改完成后，在`resul`目录下执行`go run main.go`。
5. **访问项目**：
通过地址`IP:Port`访问项目。

## English Version
### Project Introduction
This project has implemented a feature where logging in also serves as registration. Based on this, the user function module has been developed.

### Project Operation Steps
1. **Clone the Project**:
Use `git clone` to clone the project address.
2. **Modify the Configuration**:
Modify the following configurations:
    - The connection configurations of Redis and MySQL.
    - The email address and its authorization code.
3. **Initialize the Project**:
Execute `go mod tidy` to initialize the project.
4. **Run the Project**:
After the modification, execute `go run main.go` in the `resul` directory.
5. **Access the Project**:
Access the project through the address `IP:Port`.
